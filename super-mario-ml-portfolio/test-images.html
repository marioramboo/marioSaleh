Instructions

If images don't load:

  1. Make sure you're opening the HTML file through a web server (not double-clicking the file)
  2. Try using a local web server like Live Server in VS Code
  3. Or use Python's built-in server: python -m http.server 8000
  4. Then open: http://localhost:8000
